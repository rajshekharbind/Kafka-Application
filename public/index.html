📖 How to Use Kafka Pro

🔧 Cluster Information

View your Kafka cluster details including:

  • Brokers: Number of active broker nodes
  • Controller ID: Current cluster controller
  • Status: Real-time cluster health status

📑 Topics Management

Create Topics:

  1. Enter topic name (alphanumeric, dots, dashes)
  2. Set number of partitions (1-32)
  3. Set replication factor (1-3)
  4. Click "Create Topic"

Search Topics: Use the search bar to filter topics by name

Select Topic: Click any topic to auto-fill producer/consumer forms

📤 Producer (Send Messages)

  1. Enter or select a topic name
  2. (Optional) Add a message key for partitioning
  3. Enter message value (text, JSON, etc.)
  4. Click "Send Message"
  5. View character count and validation feedback

📥 Consumer (Receive Messages)

  1. Enter or select a topic name
  2. Enter consumer group name (auto-filled: default-group)
  3. Click "Start Consumer" to begin listening
  4. Messages will appear in real-time below
  5. Use search to find specific messages
  6. Click "Stop Consumer" to stop listening
  7. Click "Clear Messages" to remove all messages

💡 Tips

  • 🔄 Auto-refresh happens every 10-15 seconds
  • ⚫ Green circle = Connected, Red circle = Disconnected
  • 🔍 Use search to filter topics and messages
  • 🌙 Toggle dark mode for comfortable viewing
  • ⏱️ Timestamps show when messages were received
